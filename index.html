Project Addendum

This web page is part of the ITC 505 Lab 6 assignment, where we focused on using basic HTML, CSS, and JavaScript to create an interactive webpage. Below is a summary of key details:

Technologies Used:

  • HTML5: Used to structure the content of the page.
  • CSS3: Applied for styling the layout, including grid systems, colors, and typography.
  • JavaScript: Implemented to create an interactive "Number Guessing Game", and dynamic menu navigation using event listeners and DOM manipulation.

Project Highlights:

  • Responsive Layout: The page adapts to various screen sizes using CSS Grid layout.
  • Interactive Game: A simple number guessing game using JavaScript where the user tries to guess a number between 1 and 10.
  • Navigation System: The menu allows for smooth section navigation by hiding and showing content dynamically based on user input.

Process and Verification:

  • Menu Navigation: The JavaScript function showSection() was implemented to handle dynamic content display. Each menu item links to a corresponding section, and only the selected section is visible at a time, while others remain hidden.
  • Number Guessing Game: A random number between 1 and 10 is generated. The user's guess is compared with this number, and feedback is given (e.g., "Too low", "Too high", or "Correct"). If the user provides an invalid input, a prompt appears asking them to enter a valid number.
  • Last Updated Date: The JavaScript document.lastModified method dynamically updates the "Last updated" date at the bottom of the page to reflect when the content was last modified.

Challenges Encountered:

Ensuring smooth section transitions and proper validation of user input was challenging. The section visibility logic required careful tweaking to prevent any sections from showing at the wrong time.

Future Enhancements:

  • Add a timer and scoring system to the game for added complexity.
  • Optimize the layout further for mobile users.
  • Introduce ARIA labels and keyboard navigation for improved accessibility.
